## Iniciar o Projeto Laravel

Crie um banco de dados em seu local

Configure seu .env, inserindo o banco de dados no laravel.

Inicie a migração para o banco de dados com:

"php artisan migrate"

Depois rode o projeto com:

"php artisan serve"
